Objective of the Project

The main aim of this project is to estimate the poverty/welfare levels of the borrowers in the regions where Kiva has active loans and to identify countries and regions to which kiva can extend their services to. Factors like Multidimensional Poverty Index (MPI) and Human Development Index (HDI) were analyzed for different countries to estimate the poverty levels.

Data for Kiva Crowdfunding project is collected from the following sources:

Kiva Loans data is collected from Data Science for good kiva Crowdfunding.

Multidimensional poverty index data is collected from Multidimensional Poverty Measures.

Human development Index data is collected from United Nations Development Programme.

Data Preprocessing

Removed unwanted columns from the datasets and replaced null values in latitude and longitude columns of kiva mpi region locations dataset with the values provided by google

Merged Kiva Loans and MPI National dataset based on country column for analysing the poverty/welfare levels of the borrowers in the regions where Kiva has active loans
